INTERMEDIATE WEBDESIGN CAPSTONE PROJECT: LEARNING FROM A PROFESSIONAL WEBSITE
OBJECTIVE

To recreate a professional company website with (1) homepages and (2) inner pages based on the content structure of a Wikipedia article. Must define user segments, reconstructe the information architecture, create the structural navigation, develop prototypes and wireframes for a rational plan of each webpage using mockup tools, and build the website using Bootstrap format and CSS. For the purpose of this project I have chosen to recreate classic.motown.com for a Michael Jackson information blog.

PLAN AND RATIONALE

I reconstructed the information architecture from the content listed on the article and reorganized it as the content hierarchy of my webpage’s structural navigation. I have focused on global navigation as my website's main navigation and utility navigation as social media links. The clear and simple navigation aligns with usability and accessiblity principles following specified user personas.

RESULT

The final project consisted of a static website with multiple inner pages that were coded from scratch using HTML Bootstrap grid format and styled with CSS. I have successfully maintained the design integrity of the original website. From a static standpoint, the website I have developed is almost an exact replica of the professional Classic Motown website. A side by side comparison of Classic Motown (left) and my website (right) shows the similarity. Through this project I have become exposed to and learned the various design and technical principles associated with the visual elements of a professional website.
